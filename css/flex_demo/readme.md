- 格式上下文 CSS Format Context
    1. 新的盒子可以构建全新的显示上下文
    2. display: flex;   弹性  
        - display 声明显示上下文 (环境)
    3. 弹性布局由父元素和多个子元素组成的全新渲染环境
    4. 子元素不受换行的约束,原因flex 全新的格式上下文
    5. 按比例弹性分配空间, 每个子元素 
    flex:1   2 
        - (1 : 1 : 1)  
        - (1 : 2 : 3)
